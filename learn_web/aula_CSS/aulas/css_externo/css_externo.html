Aula CSS externo

Usado para colocar o mesmo css em várias páginas.

Modo em que normalmente as empressas trabalham.

Isso facilita a estilização do projeto, podendo ser feita em um arquivo apenas.
Usando: " link rel="stylesheet" href="./estilo.css" "

Verificar outra página:Link

Outra importante possibilidade

É costume também usar arquivos CSS diferentes para estilizar coisas diferentes em cada página:

  • Fonte
  • Cores
  • Tamanhos

Ou seja, geralmente temos múltiplos arquivos CSS em um projeto
Uns mais gerais, e outros mais específicos

Sobreposição de estilos

É possível modificar o estilo em uma página específica usando o css interno.
A ordem de prioridade de CSS´s é:

  1. CSS inline
  2. CSS interno
  3. CSS externo

Por isso o significado de CSS é FOLHAS DE ESTILO EM CASCATA

Dica: não bagunce o projeto

Procure fazer toda a estilização em um arquivo externo
