Especifidades CSS

Vimos que há uma ordem de prioridade dos comandos de estilização CSS

A ordem de prioridade de CSS´s é:

  1. CSS inline: na própria tag com "style=..."
  2. CSS interno: no head do arquivo HTMl e a classe/id/seletor na tag
  3. CSS externo: com um arquivo css a parte e a classe/id/seletor na tag

Porém, vale lembrar que identificadores (id), classes (class) e seletores também possuem prioridades (especifidades) diferentes

MAIS INFORMAÇÕES SOBRE ESPECIFIDADE AQUI!

Ordem de especifidade

A ordem CRESCENTE de especifidade no CSS é:

  1. SELETOR DE ESPECIFIDADE: p{}
  2. ELEMENTO DE UMA CLASSE: .class p{}
  3. CLASSE: .class p{}
  4. CLASSE ESPECIFICADA: p.class{}
  5. IDENTIFICADOR: #ident

Important

Important é um recurso usado que FORÇA uma atribuição ignorando efeitos de prioridade de:

  • Tipos de CSS
  • Ordem de Especifidade

Ter cuidado ao usar!

Muito bom para fazer testes de como a página ficaria com uma dada característica

Use com moderação no projeto, pois o uso em excesso torna seu uso inviável

Como usar:

" propriedade: ...!important; "
