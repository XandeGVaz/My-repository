Propriedade Text

Principais modificações

  • "text-align: center/left/right/justify..." - alinhamento do texto
  • "direction: rtl/ltr" - direcionamento do texto
  • "unicode-bidi: bidi-override" - usado para escrita de traz pra frente (idiomas asiáticos)
  • "text-decorantion: none/overline/linhe-through/underline" - decoração do texto (muito usado em links)
  • "text-transform: uppercase/lowercase/capitalize" - muito usada para pradronização de textos (tudo maiúsculo/minúsculo ou com inicial maiúscula)
  • "text-ident: --tamanho--" - efeito de parágrafo
  • "letter-spacing: --tamanho--" - efeito de espaçamento entre letras (possui valores negativos e pode ser usada para efeitos de fonte)
  • "line-height: --tamanho--" - efeito de espaçamento entre as linhas
  • "word-spacing: --tamanho--" - espaçamento entre as palavras
