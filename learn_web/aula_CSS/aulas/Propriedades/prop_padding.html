Propriedade Padding

Como funciona?

O padding funciona de forma parecida com o margin

Porém, o padding serve para modificar os espaçamentos internos
Ou seja, o padding é uma especie de PRENCHIMENTO

Possue duas possíveis formas:

  • "margin-(lado):--tamanho--"
  • "margin: --top-- --right-- --bottom-- --left--"

(Colocando os tamanhos em pixels)

Utilidade

Muito utilizado para fazer botões
