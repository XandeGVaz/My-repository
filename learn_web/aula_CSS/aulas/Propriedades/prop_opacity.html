Propriedade Opacity

Responsável pela transparência de imagens e outros elementos

Necessário ter cuidado: opacidade é passada para todos os elementos dentro da classe a qual foi atribuida

Como funciona?

"opacity: --quantidade--"
Sendo essa quantidade de 0 (mais transparente) a 1 (menos transparente)
