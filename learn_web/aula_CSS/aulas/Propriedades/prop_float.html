Propriedade Float

Como funciona

O principal objetivo do float não é o posicionamento, mas sim a criação de um efeito visual de contorno

Essa propriedade permite que um elemento se "desprenda" da div (flutue) e assuma uma posição desejada criando um efeito em relação aos elementos ao redor

Com isso, o elemento assume tal posição e os demais elementos da div se rearranjam de forma a concordar com tal posicionamento

Funciona escrevendo: float: --posição-- (top, right, bottom, left)

Usos recorrentes

Diagramação de sites/informações

A propriedade float permite a disposição de elementos de forma mais organizada e/ou visualmente bonita na página web
