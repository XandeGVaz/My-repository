Propriedade Display

FUNCIONAMENTO DETALHADO DO DISPLAY AQUI!

display:

  • block
  • inline
  • inline-block
  • flex

display: block

Permite configurações de:

  • width
  • height
  • margin

Elementos em bloco pulam linha (tem um espaço horizontal só para eles)

Parágrafos, títulos e elementos de uma lista já vem por padrão com display em bloco

Exemplo:

  • Item 1
  • Item 2
  • Item 3

display: inline

Sem as configurações de tamanho altura e margem

Cada elemento do display em linha se comporta como uma palavra
Consequentemente só terão espaços entre eles se existir um espaço ou quebra de linha no código

Span, e links já vem por padrão com display em linha

Exemplo:

  • Item 1
  • Item 2
  • Item 3

display: inline-block

Elementos aparecem em linha, mas cada um funciona como um bloco

Então, cada elemento alinhado permite a configuração de:

  • width
  • height
  • margin

Exemplo:

  • Item 1
  • Item 2
  • Item 3

display: none

Permite cancelar a aparição de um elemento

Nesse caso os outros aparecem como se ele não existisse

Exemplo:

  • Item 1
  • Item 2
  • Item 3
