Propriedades Text Shadow e Box Shadow

Text Shadow

Funcionamento:

" text-shadow: --deslocamento horizontal-- --deslocamento vertical-- --bluur(suavização)-- --cor--; "

E para combinar sombras:
" text-shadow: --deslocamento horizontal 1-- --deslocamento vertical 1-- --bluur(suavização) 1-- --cor 1-- , --deslocamento horizontal 2-- --deslocamento vertical 2-- --bluur(suavização) 2-- --cor 2--; "
