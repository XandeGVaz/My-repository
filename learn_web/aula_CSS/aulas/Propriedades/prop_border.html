Propriedade Border

Como funciona?

A atribuição de uma borda , pertencente à uma div/tag, é feita da seguinte maneira:

"border: --tamanho-- --tipo-- --cor--" (não necessariamente nessa ordem)

FUNCIONAMENTO DETALHADO AQUI!

Tipos de borda

  • solid: sólida
  • dotted: pontos
  • dashed: traços
  • double: dupla
  • groove: tridimensional
  • ridge: com topo(parece uma moldura)
  • inset
  • outset
  • none
  • hiden

Para modificar um lado específico da borda

Isso é feito da seguinte forma: "border-(lado):"
Sendo os possíveis lados:

  • left
  • right
  • top
  • bottom

Para mudar o formato de uma borda

Borda arredondada:"border-radius:--tamnho do arredondamento--"
