Propriedade Font

Principais modificações

  • "font-family:" - família da fonte
  • "font-size:" - tamanho da fonte
  • "font-style:" - style da fonte (itálico/oblíoquo etc)
  • "font-weight:" - grossura da fonte (--bold-- para negrito)
