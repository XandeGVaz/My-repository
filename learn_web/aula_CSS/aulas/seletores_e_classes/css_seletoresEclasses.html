Aula Seletores e Classes

Existem 3 tipos de seletores

  • TAG (tipo tag)
  • CLASS (classe)
  • ID (identificador)

Em geral os seletores são responsáveis por "encaminhamento" do estilo css ao lugar de interesse.

Seletor por TAG

Nesse caso colocamos no arquivo css " tag{} " , podendo ser h1, p ou até uma div.
E dentro das chaves fazemos as atribuições.

Assim, tais atribuições são feitas à todas as tags do determinado tipo.

Seletor por CLASSE

Nesse caso, colocamos no arquivo css " .--nome da classe--{}", com as atribuições dentro das chaves.

Essa classe pode ser aplicada às tags que desejamos que pertença a uma classe com um conjunto de atribuições.
Para isso, escrevemos: tag class="--nome da classe 1-- --nome da classe 2-- ..." .

Seletor por ID

Nesse caso, colocamos no arquivo css " #--nome do id--{}"
E dentro das chaves fazemos as atribuições

Vale lembrar que o ID só vale para uma TAG, funcionando como um indentificador único de uma tag.
Para atribuir o ID a uma tag: tag id="--nome do id--".

O ID nunca pode começar com um número.
