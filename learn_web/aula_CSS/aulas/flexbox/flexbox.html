Flexbox

Maneira mais atual de organizar conteúdos em divs (principalmente menus) sem utilizar:

  • Display: inline-block
  • Float: left

A flexbox facilita tratar a disposição dos elementos em dispositivos de tamanhos de telas diferentes, sem ter que mudar a estrutura dos códigos totalmente para diferentes media queries

Código de uma flexbox:

.flexpai :

/*Principal:*/
display: flex

/*Direcionamento dos itens*/
flex-direction: column(itens separados em blocos)/ row(um item do lado do outro)
+ -reverse(troca ordem de exibição dos elementos)
/*Comportamento dos itens ao exceder div*/
flex-wrap: nowrap(elementos saem da div pai)/ wrap(ao atingir limite da div pai, elementos saão jogados para baixo)

/*2 últimos podem ser substituídos por:*/
flex-flow: row/column wrap/nowrap

/*Posicionamento dos itens, da flexbox, na horizontal*/
justify-content: center/ flex-end(direita)/ flex-start(esquerda)/ space-between (itens com espaçamento igual)/ space-around(itens com espaçamento igual em relação a eles e o entorno)

/*Posicionamento dos itens (individualmente), da flexbox, na vertical*/
align-items: center/ flex-end(baixo)/ flex-start(cima)

/*Posicionamento dos itens (em conjunto), da flexbox, na vertical*/
align-content: center/ flex-end(baixo)/ flex-start(cima)

.flexpai .filhos:

/*Crescimento automatizado dos filhos,se adaptando ao tamanho da tela*/
flex-grow: --ritmo de crescimento-- /* posso colocar essa propriedade individualmente no style de cada filho, assim terei elementos de tamanhos diferentes na tela*/

/*Tamanho responsívo específico de cada filho em relação ao tamanho total da flex*/
flex: --tamanho--% /*sequencialmente os elementos ocuparão a mesma linha até atingir os 100% de largura da tela*/
/*É necessário para isso considerar margem entre os itens e também adicionar no início do arquivo:*/
*{
box-sizing: border-box;
}
