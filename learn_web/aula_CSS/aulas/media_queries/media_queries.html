Media Queries

Qual a função

Media queries permitem fazer mudanças na página dependentes do tamanho de tela do usuário

Consequentemente, diferentes dispositivos possuirão diferentes visualizações do site

GRANDE IMPORTÂNCIA: permite portabilidade da página para diferentes dispositivos

Como definir um Media Querie?

@media only screen and (max-width:--tamanho máximo da tela--) and (min-width:--tamanho mínimo da tela--){
(comportamento desejado para essa faixa de tamano)
}

IMPORTANTE

PADRÃO DE MEDIA QUERIES PARA DIFERENTES DISPOSITIVOS:

/*Smartphones (600px para baixo)*/
@media only screen and (max-width: 600px) {...}

/*Dispositivos um pouco maiores, mas ainda pequenos (Pequenos Tablets e Smartphones + largos, 600px para cima)*/
@media only screen and (min-width: 600px) {...}

/*Dispostivos Médios (Tablets deitados, 768px para cima)*/
@media only screen and (min-width: 768px) {...}

/*Dispostivos Largos (laptops/desktops, 992px para cima)*/
@media only screen and (min-width: 992px) {...}

/*Super Largos (Telas maiores laptops, desktops e até TVs, 1200px para cima)*/
@media only screen and (min-width: 1200px) {...}
