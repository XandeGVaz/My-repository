Sintaxe do JS

Sintaxe Básica

Sintaxe básica parecida com o da linguagem C

  • ("") para strings, e (--exp--) para expressões com variáveis/números
  • É case sensitive
  • Não aceita hífen em nomes de variáveis
  • Comentação (// e /**/)

Variáveis

Grande diferença do C nos tipos de variáveis

As possíveis variáveis do JS:

  • var
  • let
  • const

VAR

Coringa: permite reatribuição e redeclaração independente da hierarquia dos blocos

LET

Permite reatribuição

Não permite redeclaração dentro do mesmo bloco

Mas caso mudemos de escopo, a redeclaração funcionará dentro desse escopo

CONST

Não permite reatribuição e nem redeclaração independente do bloco atual

Concatenação

O JS permite concatenação de variáveis, permitindo unir strings

Não é aconselhável concatenar strings juntamente a números
Mas caso faça isso, o primeiro elemento a ser concatenado determinará se será uma concatenação ou uma soma de elementos

/*No arquivo js dessa aula*/

let a, b, c;
a = "hello"; b = "world";
c = a + " " + b + "!";

/*resultado:*/

Funções

Funcionamento parecido com o do C, mas sem a tipagem dos parâmetros

Exemplos

No arquivo JS:

/*Função de dar oi para uma pessoa com um dado nome*/
function hello(name){
alert("hello " + name);
}

Caso necessário, a funções podem ter um retorno

/*Função de soma*/
function soma(a, b){
return a+b;
}
