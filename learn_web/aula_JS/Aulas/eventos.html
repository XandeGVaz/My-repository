Eventos JS

O que são?

Ações disparadas pela interação do usuário com a página

Existem muitos eventos. Os mais utilizados são:

onclick

ondbclick

onmouseover, onmouseout, onmousemove

onmousedown e oumouseup

Clique, segure e solte

onfocus, onchange, onblur

Servem para tags do tipo input

onkeydown, onkeypress e onkeyup

Também servem para tags do tipo input

onload e onresize

Atribuídos ao corpo da página
