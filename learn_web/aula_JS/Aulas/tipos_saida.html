Tipos de saída do JS

Existem 4 tipos principais de saídas no Java Script

InnerHTML

Neste tipo de sáida é obtido um elemento do html (uma tag por exemplo), e escreve-se algo nela pelo arquivo JS

Mais utilizado para odicionar interatividade à página

Exemplo

No arquivo html:

--p id="text"-- --/p--

No arquivo JS:

document.getElementById("text").innerHTML="Hello World!"

Document.Write

Funciona colocando dentro de um script presente no arquivo HTML (dentro de uma tag)

Já aceita funções lógicas

Saída é visualizada na própria TAG

Mais utilizado para testes

Exemplo

No arquivo html:

/*Dentro da Tag*/

document.write("text")

/* OU */

document.write(function)

Alert

Funciona tanto com texto quanto com funções

Pode ser colocado tanto no arquivo JS quanto em um script interno

Mais utilizado para obter resultados ao longo do desenvolvimento

Exemplo

No arquivo JS anexado:

alert("text")

/* OU */

alert(function)

Console.Log

Mais silencioso, aparecendo somente na área de CONSOLE do site

Permite olhar respostas do navegador sem atrapalahr fluxo da página

Para visualizar: apertar F12

Mais utilizado para obter resultados ao longo do desenvolvimento

Exemplo

No arquivo JS:

console.log("text")
