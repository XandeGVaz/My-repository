Objetos JS

O que são?

Objetos são variáveis que possuem:

  • Propriedades (valores) na forma de atributos
  • Métodos que são funções dentro de propriedades

Exemplo

Classe carro

const carro = {

marca: "ford",
modelo: "ka",
ano: 2015,
placa: "ABC-1234",

buzina: function(){ alert("BIIIIIIIII")},
completo: function(){
return "O carro é da marca " + this.marca + " e do modelo " + this.modelo;
}

};
